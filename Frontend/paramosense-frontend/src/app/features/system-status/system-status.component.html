Preguntas frecuentes

¿Por qué no veo datos en el dashboard? Verifica que el nodo Arduino esté encendido y conectado vía LoRa al concentrador. El concentrador debe tener WiFi activo y publicar en el broker MQTT.
¿Con qué frecuencia se actualizan los datos? Los sensores envían lecturas cada 30 segundos. El dashboard se refresca automáticamente.
¿Cómo agrego un nuevo sensor al mapa? Accede al Panel de Administrador con la contraseña de operador y usa "Agregar nodo real" para vincular un nodo Arduino activo.
¿Qué significa el estado "Crítico" en una alerta? El sensor registró un valor fuera del rango seguro para el ecosistema paramuno (temperatura >20 °C, humedad <40% o nivel de agua <40%).
